The "Before" folder contains a version of the ChatCave iOS project before any
integration with NSURLSession.

The "After" folder contains a version of the ChatCave iOS project after the
basic integration with NSURLSession that relies on HTTP cookies for
authentication.

The "After-BasicAuth" folder contains a version of the ChatCave iOS project
that uses HTTP Basic Authentication with NSURLSession.

The "After-CustomAuth" folder contains a version of the ChatCave iOS project
that uses custom authentication via HTTP headers along with NSURLSession.
